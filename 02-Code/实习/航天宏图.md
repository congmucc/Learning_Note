- 备份标准
  - 文件修改：使用`mv`命令将文件名字设置为`.bak`形式。 例如：`mv xjar.go xjar.go.bak`
  - 内容修改：将文件命令先注释，之后进行修改。
- 回滚标准是使用`cp`命令将文件名字后面的`.bak`删除即可 例如：`cp xjar.go.bak xjar.go`
- 使用 `前者 > 后者` 命令生成一个日志。
